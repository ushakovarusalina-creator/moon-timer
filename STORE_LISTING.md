Minimalist focus timer with project tracking. Runs in the background while you work.

## Полное описание

The Moon Timer is a distraction-free focus timer that helps you track how much time you spend on each project.

**How it works:**
Start the timer when you begin working. It counts up from zero — no pressure, no countdown anxiety. When you pause, the time is saved to your active project. Switch between projects anytime and the timer instantly shows that project's total accumulated time.

**Features:**
• Stopwatch-style timer that counts up from 0:00
• Create and manage multiple projects
• Time is tracked and saved per project
• Visual progress ring fills over 60 minutes
• Runs in the background — timer keeps going when the popup is closed
• Reset time per project individually
• Work / Break modes
• Desktop notifications after 60 minutes of focus
• Minimal dark UI with a calming moon aesthetic

**Privacy:**
All data is stored locally on your device using Chrome's session storage. Nothing is sent to any server.
